HTML

HTML skapades år 1991 av Tim Berners Lee som var en del av gruppen Internet Engineering Task Force (IETF) ungefär samtidigt som han kom på namnet world wide web (www). Målet med hyper text markup language (HTML) var väldigt simpelt till en början, att skapa skrifter som var åtkomliga via det då nya internet men hans version av html kom inte att släppas till allmänheten förräns år 1995 och kom då att kallas för html 2.0.

HTML 1.0

släpptes år 1993 med syftet att skapa texter som var åtkomliga via webben/internet men på grund av att många av dåtidens webbutvecklare inte hade någon input i utvecklings processen så blev det ingen stor succé.

HTML 2.0 innehöll alla funktioner hos HTML 1.0 men med ny funktionalitet. Detta ver den första stora uppdateringen till HTML. Nya funktioner och tweaks kom med varje ny iteration av html men den största som kom efter detta var HTML 4.0

HTML 4.0 kom med massa ny funktionalitet inklusive stöd för Cascading style sheets (CSS) som kom att bli en standard för hemsidor tillsammans med HTML 4.0 och senare 4.01 vilket i princip var samma sak men mindre buggig och bättre optimerad. HTML 4.0 släpptes i december år 1997 och uppdateringen 4.01 kom samma månad som version 4.0 två år senare. Det vill säga 1999 i december.
